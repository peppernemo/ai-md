# second-brain-md

A companion site for *Build Your Own AI Workflow* — a talk on running a personal
[Claude Code](https://docs.anthropic.com/claude-code) "second brain" for clinical,
academic, and administrative work, built on one rule: **no PHI ever enters the LLM.**

Live: https://peppernemo.github.io/second-brain-md/

Contents:
- The PHI rule + a `.gitignore` that keeps patient data out of version control
- A copy-paste **Initialize Prompt** that scaffolds the whole workspace
- A stripped-down clinician `CLAUDE.md`
- The PHI guardrail block + example skills

By Janos Barrera, MD — Hand & Orthoplastic Surgery. Built for the NYU Plastic Surgery Residency.
